## PSO-16C-001

Origin:
Engine v0.1

Observation:

Existence and Discoverability are independent concepts.

A path may exist even if no observer discovers it.

Implication:

Topology and Observation must remain separate ontological layers.

Origin:
VISUAL TEST 002

Observation:

The Engine distinguishes:

Reachable + Discoverable

from

Reachable + Not Discoverable

but the current Visualizer renders both as:

Reachable

when can_reach = True.

Implication:

Engine State and Visual Representation are distinct layers.

This is a visualization observation,
not an ontology observation.
